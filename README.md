# ChineseOrator
A program that will orate selected chinese characters on a Macbook

## Usage

Compile the Swift script with `swiftc ChineseOrator.swift -o ChineseOrator -framework AppKit -framework ApplicationServices -framework Carbon -framework AVFoundation`

Add to LaunchAgents with 'AddToLaunchAgents.sh'

Use `Control + Option + Command + O` to read out the highlighted chinese characters using the voice specified in Apple Settings
